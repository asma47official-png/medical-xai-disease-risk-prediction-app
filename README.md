# medical-xai-disease-risk-prediction-app
Deep learning-based medical risk prediction app using ANN. It predicts disease risk from age, BMI, and glucose level. Built with TensorFlow, Keras, and Scikit-learn, and deployed using Gradio for real-time, user-friendly predictions with high or low risk output.
This project is a Deep Learning-based medical risk prediction system designed to estimate disease risk using basic patient health indicators. It demonstrates how Artificial Neural Networks (ANNs) can be applied in healthcare analytics to support early risk detection and decision-making.
The system takes structured health inputs such as age, BMI, and glucose level, and processes them through a trained neural network model built using TensorFlow and Keras. To improve generalization and reduce overfitting, the model incorporates regularization techniques including Dropout layers and L2 regularization. Data preprocessing steps such as normalization and feature scaling are applied using Scikit-learn to ensure consistent model performance across different input ranges.
A key highlight of this project is its interactive web application built with Gradio. The web interface allows users to input patient data easily and receive instant predictions in real time. The model outputs a classification of either high disease risk or low disease risk, making it simple to interpret even for non-technical users.
This project is structured to bridge the gap between machine learning research and practical deployment. It not only focuses on model accuracy but also on usability and accessibility through a lightweight web interface. The Gradio app makes it possible to deploy the model quickly without complex backend infrastructure
Technologies used in this project include Python, TensorFlow/Keras, Pandas, NumPy, Scikit-learn, and Gradio. These tools work together to handle data processing, model training, evaluation, and deployment.
The objective of this project is to demonstrate how AI and deep learning techniques can be used in healthcare applications for early risk assessment. It serves as a strong foundation for further research in medical AI, explainable AI (XAI), and real-world clinical decision support systems.
